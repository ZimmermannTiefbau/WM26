# Zimmermann Tiefbau WM Tippspiel Online

## Dateien
- index.html
- styles.css
- config.js
- app.js
- supabase_setup.sql

## Ablauf
1. Supabase Projekt anlegen.
2. Supabase → SQL Editor → supabase_setup.sql komplett ausführen.
3. Supabase → Project Settings → Data API:
   - Project URL kopieren
   - anon public key kopieren
4. In config.js eintragen:
   - SUPABASE_URL
   - SUPABASE_ANON_KEY
5. Alle Dateien in ein GitHub Repository hochladen.
6. Vercel → Add New Project → GitHub Repository importieren → Deploy.
